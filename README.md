# TP 1 : Déploiement d'un microservice avec Docker

Le fichier app.py contient une application Flask qui renvoie le message 
'Hello from Dockerized Microservice!' quand on se rend sur l'url http://localhost:5000

Le fichier requirements.txt contient les dépendances pour faire fonctionner l'application

Le fichier Dockerfile contient les étapes pour construire l'image Docker de l'application
